# 1.3 Java Collections - Nivel 1

## 📄 Descripción - Enunciado del ejercicio

Este proyecto contiene la implementación de tres ejercicios de la Tasca 03 - Collections.

---

## 💻 Tecnologías Utilizadas

- **Java 17** (JDK 17 o superior)
- **IntelliJ IDEA** - Entorno de desarrollo integrado
- **Git** - Control de versiones
- **Java Collections Framework**:
  - `ArrayList`
  - `HashSet`
  - `HashMap`
  - `Iterator` / `ListIterator`
- **Java I/O**:
  - `BufferedReader` / `FileReader`
  - `BufferedWriter` / `FileWriter`

## ▶️ Ejecución

### Ejercicio 1: Month (ArrayList y HashSet)

1. Navega a `src/exercici1/Main.java`
2. Haz clic derecho sobre el archivo y selecciona `Run 'Main.main()'`
3. Observa la salida en la consola:
   - Listado de meses sin "August"
   - Listado con "August" insertado en la posición correcta
   - Recorrido con `for` y con `Iterator` del `HashSet`

### Ejercicio 2: List Inversion (ListIterator)

1. Navega a `src/exercici2/Main.java`
2. Haz clic derecho sobre el archivo y selecciona `Run 'Main.main()'`
3. Observa la salida:
   - Lista original: `[1, 2, 3, 4, 5]`
   - Lista invertida: `[5, 4, 3, 2, 1]`

### Ejercicio 3: Juego de Capitales (HashMap y Archivos)

1. Navega a `src/exercici3/Main.java`
2. Haz clic derecho sobre el archivo y selecciona `Run 'Main.main()'`
3. Sigue las instrucciones en la consola:
   - Introduce tu nombre
   - Responde a las 10 preguntas sobre capitales de países
   - Al final, verás tu puntuación
4. Verifica que se ha creado el archivo `clasificacio.txt` en la raíz del proyecto con tu resultado
